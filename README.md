# Buscador de Cursos da Alura

Este é um projeto em PHP que busca cursos no site da Alura utilizando as bibliotecas **GuzzleHttp** e **Symfony DomCrawler**.

## Requisitos

- PHP 8.2 ou superior
- Composer
